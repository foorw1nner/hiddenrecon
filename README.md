```bash
▶ git clone https://github.com/foorw1nner/gf-addparams.git
▶ cd gf-addparams
▶ chmod +x gf-addparams
```

Usage
```bash
./gf-addparams [flags] [wordlist] [json file name]
```

Example
```bash
./gf-addparams -iE parameters xss.json
```
