Installation
```bash
▶ git clone https://github.com/foorw1nner/hiddenrecon.git
▶ cd hiddenrecon
▶ chmod +x hiddenrecon.sh
```

Usage
```bash
[buffer] | ./hiddenrecon.sh [flags]
```

Example
```bash
./hiddenrecon.sh -eih -eda
```
